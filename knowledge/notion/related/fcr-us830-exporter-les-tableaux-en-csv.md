# User Story
**En tant que** admin ou gestionnaire,
**je veux** exporter les données d'un tableau en CSV,
**afin de** les analyser ou les partager.

---


# Périmètre
- Bouton "Exporter" sur tous les tableaux (bâtiments, lots, tiers, missions)
- Exporte les colonnes visibles avec les filtres actifs
- Format CSV (UTF-8 BOM pour Excel)
- Accessible admin et gestionnaires
