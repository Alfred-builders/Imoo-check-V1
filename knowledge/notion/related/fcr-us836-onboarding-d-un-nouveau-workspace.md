# User Story
**En tant que** nouvel admin,
**je veux** être guidé lors de la configuration initiale de mon workspace,
**afin de** démarrer rapidement sans me perdre dans les paramètres.

---


# Périmètre (à cadrer)
- Wizard en 3-4 étapes après la première connexion :
  1. Informations workspace (nom, type, logo, adresse, SIRET)
  1. Branding (logo, couleur primaire)
  1. Import optionnel (CSV bâtiments/lots)
  1. Invitation premiers utilisateurs (email + rôle)
- Chaque étape est skippable
- Le wizard peut être relancé depuis les Paramètres
- Checklist de progression visible sur le dashboard tant que l'onboarding n'est pas complété

---
